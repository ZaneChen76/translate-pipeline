# translate-pipeline CLI Usage

```
usage: translate_pipeline [-h] {translate,quality-report} ...

TranslateDocs - 技术文档/标书中译英保真翻译系统

positional arguments:
  {translate,quality-report}
                        Commands
    translate           Translate a document
    quality-report      Generate visual quality dashboard for one source and
                        one/multiple targets

optional arguments:
  -h, --help            show this help message and exit

```
